# EnumSelector

EnumSelector can be wrapped around any enum and provides a select for it. As a second argument a function
is passed that maps an enum value to a translation key.

```ts
enum Alphabet {
    A,
    B,
    C,
}

const AlphabetSelector = EnumSelector(Alphabet, (key) => `translate.key.${key}`);
```
